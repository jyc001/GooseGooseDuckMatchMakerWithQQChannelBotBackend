# Goose Goose Duck MatchMaker With QQ Channel Bot Backend
## A Goose Goose Duck Matchmaker with an official QQ Channnel bot. This is the backend project.
### mainly used libraries
- express

## How to run
### develop
````bash
npm watch
````
##### or with yarn
````bash
yarn watch
````
### run
````bash
npm start
````
##### or with yarn
````bash
yarn start
````
